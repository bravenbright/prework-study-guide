Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • A margin indicates how much space we want around the outside of an element
  • padding indicates how much space we want around the content inside and element.

CSS

  • The padding property adds space around the content inside an element. When you increase padding, you are actually making the entire element bigger

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: command to commit all files in the working branch
  • git commit -m : this commits changes and the '-m' is a flag for the descriptive message
  • git pull origin main : pulls base brach into feature before pushing code up
  • git push origin feature/add-html
  • The easiest way to switch branch on Git is to use the “git checkout” command and specify the name of the branch you want to switch to. A quick way of switching branch on Git is to use the “git switch” command and specify the name of the branch you want to switch to.
